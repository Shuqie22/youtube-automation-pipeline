# YouTube True Crime Channel Automation

Built by Sukurah Yusuf

## About This Project

This repository contains a fully automated YouTube content pipeline built with n8n that produces complete, ready-to-upload true crime videos from a single topic input. The workflow handles AI script generation, voiceover production, image animation, video assembly, caption burning, and YouTube upload without any manual editing work in between.

## Repository Structure

The workflow files and documentation are inside the `youtube-true-crime` folder. Open it to find the full README explaining how the system works, the tools used, and how to set it up.

[View the full project](./youtube-true-crime)

## Tech Stack

n8n, NCA Toolkit, ElevenLabs TTS, OpenAI, MinIO S3, ffmpeg, YouTube Data API v3, Google Sheets
